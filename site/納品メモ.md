## 技術メモ

- **リンクはすべてルート絶対パス**（/company/ 形式）。ローカル確認はドキュメントルートを本フォルダにしたHTTPサーバで（例: `python -m http.server`）
- **GTM: GTM-5CRJNVT3** を全ページの `<head>` 冒頭＋`<body>` 直後に挿入済み（academy_cp と同一）
- 以下のリンク先は**既存ページをそのまま利用**（本納品物には含まない）: /academy_cp/、/gogo/、/top-sales/、/line-optin/、旧News記事、https://digipara.comix.co.jp/
- 決算公告のPDF等、wp-content 配下のファイルへのリンクは現行URLのまま
- フォント: Google Fonts（Noto Sans JP）をCDN読み込み
- 文字サイズポリシー: 最小14px（注記・コピーライトのみ13px）
- レスポンシブ: フルナビは1000px以上、未満はハンバーガー。〜768pxでモバイルレイアウト
- FV・ABOUT等の画像はAI生成のオリジナル（権利問題なし）。ロゴ類は現行サイトから取得した公式素材

## SEO実装済み項目（2026-07-05追加）

- **canonical**: 全ページ（404除く）に `https://www.comix.co.jp/...` の正規URLを設定済み。ドメインが変わる場合は要書き換え
- **構造化データ（JSON-LD）**: TOP=Organization＋WebSite、下層=BreadcrumbList（表示中のパンくずと同一構成）、ニュース記事=NewsArticle。記事詳細をWPテンプレート化する際は headline / datePublished / image を記事ごとに差し込むこと
- **OGP**: og:description は各ページの meta description と同一内容に統一済み。og:image はTOP系=hero-c3.png、記事=記事画像
- **twitter:card**: summary_large_image を全ページに設定
- **フッターYouTubeリンク**: 全ページのフッターロゴ下にピル型ボタンで設置（https://www.youtube.com/channel/UCH-ji0wfvCy5zbeA0-DGypg）。表記は「中小企業のAI顧問チャンネル」（改名予定のチャンネル新名称）。Organization の sameAs にも同URLを記載
- 画像 alt は全数設定済み。見出しは各ページ h1 一意
